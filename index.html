Hourglass

This page is still under construction but it includes downloads for the source code and designs for the Hourglass chess clock. The code is complete but messy and should be considered in beta until documentation and feature implementation is complete.

Features

The Hourglass is usable on both Mac and PC (and probably Linux although I haven't built it) in XSplit and OBS. It uses Processing for the software running on the streaming computer and an Arduino as a micro controller for the clock itself.

Current users of the Hourglass are Privateer Press and Chain-Attack.

The code and design is being released in the hope that this kind of technology can support streaming tabletop miniatures games.

Don't want to build your own?

Contact me if you want an Hourglass. I build them custom to the needs of the streamer.
